# Out-of-the-box Vuetify navigation menu component, with Toolbar, Navigation Drawers, Menus, and QR code

This Package is the successor of [vuetify-nav](https://github.com/UedaTakeyuki/vuetify-nav) for **Vue3 + Vuetify3**.  
For **Vue2 + Vuetify2**, package  [vuetify-nav2](https://www.npmjs.com/package/vuetify-nav2) is avallable.

## Project setup

## Why vuetify-nav3
The **vuetify-nav3** is an out-of-the-box navigation menu component **without messy boilerplate** codes.

## Install
### npm or yarn

```bash:
npm install vuetify-nav 

# or

yarn add vuetify-nav 
```

For detailed setup, refer to the [wiki](https://github.com/UedaTakeyuki/vuetify-nav3.1/wiki/Set-up-for-npm-or-yarn).

### cdn

```
    <!-- vuetify-nav3-->
    <script src="https://cdn.jsdelivr.net/npm/vuetify-nav3/dist/VuetifyNav3.umd.js"></script>
```

For detailed setup, refer to the [wiki](https://github.com/UedaTakeyuki/vuetify-nav3.1/wiki/Set-up-for-for-cdn).
